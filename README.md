# Teste Prático QA - Automação Selenium Java

## Ferramentas Utilizadas
- Java 17+
- Maven
- Selenium 4.x
- TestNG
- WebDriverManager

## Pré-requisitos
- Java instalado
- Maven instalado
- Google Chrome instalado

## Como Executar
```bash
mvn clean test
```

## Estrutura
- `pages`: Page Objects
- `base`: Configuração WebDriver
- `tests`: Testes automatizados

## Observações
- Ajustar seletores conforme HTML real do sistema.
